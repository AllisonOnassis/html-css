Símbolos e Emojis

É preciso utilizar caracteres especais para fazer símbolos aparecerem na tela, pois se colocar diretamente o simbolo, como por exemplo, os sinais de colchetes, o navegador interpreta como uma tag.

Símbolos que podem ser exibidos usando códigos são chamados de HTML Entities.

  • Espaço em branco =  
  • Marca registrada = ®
  • Copyright = ©
  • Trade Mark = ™
  • Euro = €
  • Libra (pound) = £
  • Yen = ¥
  • Cent = ¢
  • Vazio = ∅
  • Soma = ∑
  • Delta = Δ
  • Seta esquerda = ←
  • Seta acima = ↑
  • Seta direita = →
  • Seta baixo = ↓
