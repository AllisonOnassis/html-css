HTML

HTML é uma abreviação de Hpertext Markup Language, em português significa Linguagem de Marcação de Hipertexto. A base do HTML são as marcações, conhecidas como tags.

Explicando de forma simples: o HTML está relacionado ao conteúdo do site.

As Tags

Uma tag é um grupo de palavras que fica entre os colchetes angulares, a maioria possui tag abertura e uma de fechamento(indicadas com a barra), dentre elas é colocado o conteúdo.

Podem ser adicionados atributos e valores, isso configura o comportamento. Por Exempo: se tivermos uma tag a (âncora) precisamos colocar um parâmetro href="destino.html" com um valor dentro das "", que seria o destino da âncora, depois inserimos o conteúdo e colocamos uma tag de fechamento.
Exemplo de link

A mesma tag pode ter diversos paâmetro, cada um com seu valor. Porém, algumas, não tem necessidade de conteúdo interno e devido a isso não fechamento, como as tags br e img.

CSS

O CSS é uma abreviação para Cascading Style Sheets (Folhas de Estilo em Cascata). É utilizado para configurar um resultado visual dos elementos HTML, por exemplo, com as folhas de estilo conseguimos alterar a cor de fundo de um elemento, fonte, colocar borda etc.

As configurações das folhas de estilo são feitas utilizando seletores.

Vamos configurar esse paragráfo, para isso utilizamos o seletor p , depois fazemos uso de chaves para delimitar todas as declaraçoes relativas ao seletor atual e colocar pont e vírgula para indicar que a declaração se encerrou.

Todas as propriedades ter ser valor, este por sua vez deve ser separados por dois pontos.

  • fonte em Ariel
  • tamanho da letra
  • cor da letra
  • cor de fundo
