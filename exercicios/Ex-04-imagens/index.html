Existem diversos formatos de imagem, mas os mais os formatos compactados mais usados para sites são: JPEG e PNG.

O JPEG foi criado por Eric Hamilton em 1983. É um formato utilizado para gerar imagens fotográficas com um tamanho extremamente reduzido.

JPG(em formato JPEG) é usado para gerar arquivos muito pequenos que ocupam pouco espaço em disco.

O PNG (Portable Network Graphics) foi criado em 1996 pelo W3C para substituir o fomato GIG, também é um formato compactado, tem uma capacidade de configurar a opacidade de cada pixel (deixa tranparente ou com tranparencia limitada).

Na hora de criar um site opte pelo JPEG com uma compactação entre 30% e 50%. Utilize o PNG quado precisar de tranparência na imagem.
